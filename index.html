My Work

Here are some of the projects where I’ve applied software development principles to design, build, and optimize applications that solve real-world problems.

Smart CV Analyzer

View Project

Designed and implemented an AI-powered Smart CV Analyzer that automates resume analysis and job-role prediction using machine learning and NLP techniques. The system allows users to upload PDF resumes, extracts and preprocesses unstructured text, and predicts the most suitable job category using TF-IDF vectorization and supervised classification models. Based on the predicted role, the application dynamically surfaces relevant job opportunities through real-time job search APIs and recommends curated interview and career guidance videos to support user preparation. Built using Python and Streamlit, the solution delivers a clean, interactive experience while significantly reducing manual screening effort and helping users gain clear, data-driven insights into career alignment and next steps.

CyberText Shield

View Project

I designed and developed CyberText Shield, a text analysis and content-safety solution focused on detecting and classifying harmful or unsafe text using machine learning and NLP techniques. The core classification model was trained in Python using structured text datasets to accurately identify different categories of content, while the application layer was implemented using JavaScript to deliver a responsive and accessible user experience. The system processes user-provided text in real time, applies the trained model for classification, and presents clear, actionable results through a clean interface. This project highlights my ability to bridge machine learning model development with practical application development, demonstrating an end-to-end understanding of NLP workflows, cross-technology integration, and building scalable, real-world solutions.
